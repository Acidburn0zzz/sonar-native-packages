YUM

sudo wget -O /etc/yum.repos.d/sonar.repo http://home.godin.net.ru/sonar-pkg/rpm/sonar.repo
Install Sonar: 
yum install sonar

Zypper

Add repository: 
zypper addrepo http://home.godin.net.ru/sonar-pkg/rpm sonar
Install Sonar: 
zypper install sonar
